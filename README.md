## dPhone 预购智能合约

开发语言：

| 语言    | 版本   |
| ------- | ------ |
| Python  | 3.9.x  |
| Brownie | 1.19.x |



已实现功能：

| 功能             | 说明                   |
| ---------------- | ---------------------- |
| 可升级合约       | 使用透明代理           |
| 钱包多签         | 指定的治理方法需要多签 |
| 支持存入多种资产 | 包括原生资产ETH        |
| 预购失败         | 所有资产原路返回       |
| 预购成功         | 所有资产转移到指定地址 |



未确定需求的功能

| 功能                   | 说明                           |
| ---------------------- | ------------------------------ |
| DPN奖励功能            | 奖励方案未确定                 |
| 支付尾款的功能         | 预购成功后，需要支付余下的款项 |
| 用户提交和加密收货地址 | 避免敏感信息暴露               |
